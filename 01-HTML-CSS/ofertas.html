Ingeniero de Software Senior

Tech Solutions Inc. - Remoto

Descripción del puesto

Tech Solutions Inc. está buscando un ingeniero de software senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendrá una sólidad formación en desarrollo de software, con una experienca en el diseño, desarrollo e implementación de soluciones de software escalables y de alto rendimiento.

Responsabilidades

  • Diseñar, desarrollar y mantener aplicaciones web utilizando tecnologías modernas
  • Colaborar con equipos de producto y diseño para definir y entregar nuevas características.
  • Escribir código limpio, eficiente y bien documentado
  • Realizar revisiones de código y prorporcionar retroalimenatción constructiva a los miembros del esquipo

Requisitos

  • Licenciatura en informática o campos asociados.
  • Mínimo 5 años de experiencia en desarrollo de software
  • Experiencia con frameworks de javaScript (por ejemplo, React, Angular, Vue.js)
  • Familiaridad con metodolofías ágiles y herramientas de contros de versiones (por ejemplo git)

Acerca de la empresa

Tech Solutions Inc. es una empresa de tecnología innovadora que se centra en la cración de soluciones de software a la vanguardia para diversas industrias. Estamos comprometidos con el fomento de un entorno de trabajo colaborativo e inclusivo donde cada empleado pueda prosperar y crecer profesionalmente. Ofrecemos salarios competitivos, beneficios integrales y oportunidades de desarrollo profesional continuo.
